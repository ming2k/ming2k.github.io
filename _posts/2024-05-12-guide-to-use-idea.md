## Trouble

### linux 版本 ibus 候选框错位问题

更新 idea 的 JBR 版本

1. `ctrl` + `shift` + `a` 
2. find and select `Choose Boot Java Runtime for the IDE`
3. choose new jbr version, it is okay for `17.0.11+1-1207-20-jcef`

https://www.jetbrains.com/help/idea/switching-boot-jdk.html
