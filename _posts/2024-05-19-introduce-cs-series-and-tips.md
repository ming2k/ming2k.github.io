# 介绍CS系列以及技巧

CS（Counter Strike）系列很多。

## CS系列

### CS（CS 1）

CS目前停留在1.6版本。

### CS:Condition Zero

在CS基础上添加了玩法，支持单人游戏。

### CS:Condition Zero Deleted Scene

CS: CZ 的删减版本。

### CS:Source

Source引擎写的版本，画质更好。

### CS 2（之前叫 CS:GO）

目前主流多人联机CS。

## 技巧

CS:CZ 里面切枪需要按下数字并鼠标左击，可以通过下述方式快速切枪：

Enable console first, if it is disabled. Start the game with " -console" command and press ~ in the menu. Then type in: "hud_fastswitch 1"
