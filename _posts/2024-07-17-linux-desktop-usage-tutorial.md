# Linux Dektop Usage tutorial

## MINE

How does desktop know which application to open the with specified suffix name?

MIME: Multi-purpose Internet Mail Extensions.

you can change the MIME config by command like:

```sh
xdg-mime default <desktop_file_name> application/x-navi-animation
# for example:
# xdg-mime default org.inkscape.Inkscape.desktop application/x-navi-animation
```
